### 2. State-of-the-Art Mutation Algorithms (2025–2026 Survey)

All techniques below were selected because they map **directly** to Pathogen’s strengths (local MLX inference for fitness, declarative SKILL.md for strategy storage, Sentinel catalog as seed corpus).

#### 2.1 Genetic / Evolutionary Algorithms (Best Fit)
- **FORGEDAN** (arXiv 2511.13548, 2025): Genetic algorithm with multi-level perturbations (character, word, sentence). Uses RoBERTa-style semantic fitness (embedding cosine similarity + bypass success). Achieves >90 % ASR on aligned models.
- **NAAMSE** (arXiv 2602.07391, Feb 2026): Single-agent evolutionary framework. Genetic prompt mutation + hierarchical corpus exploration. Fitness = asymmetric behavioral scoring (target LLM response analysis). Open-source at https://github.com/HASHIRU-AI/NAAMSE.
- **AutoDAN-Turbo** (2025): Lifelong agent that autonomously discovers, stores, and evolves jailbreak strategies. Removes human seed requirement.
- **LLM-Virus EA** (2026): Mutation + crossover + fitness. Operators: insertion/deletion/substitution, paraphrasing, scenario-shift flips.
- **STAR** (OpenReview 2025): Shifts from text space to **latent activation space**. Extracts strategy components (principal components) and recombines them → novel strategies translated back to prompts.

**Why perfect for Pathogen**: Use MLX Llama 3.2 4-bit as the **fitness oracle** inside the AnalystNode. Store evolved “strategy chromosomes” in SQLite (extend EXPLOITATION_CATALOG).

#### 2.2 Fuzzing-Inspired Mutation (High Throughput)
- **PROMPTFUZZ** (arXiv 2409.14729, updated 2026): Two-stage fuzzing (prepare: seed selection + few-shot examples; focus: mutation with collected examples). Generates diverse, high-quality injections.
- **GPTFuzzer** (2025): Mutates seed jailbreak prompts using LLM-guided operators. Judgment model scores success. Extremely high ASR across open/closed models.
- **RedHit** (2025): Formulates injection as tree search; mutates branches adaptively.

**Operators proven effective** (combine any/all):
- Character-level: homoglyphs, zero-width, RTL overrides (`\u202E`), invisible chars.
- Word/sentence: synonym substitution, paraphrasing, payload splitting + reassembly.
- Encoding: base64, rot13, multilingual (Chinese/Arabic wrappers), syntax injection.
- Structural: scenario-shift (e.g., “You are now in developer mode”), multi-turn escalation (Crescendo style).

#### 2.3 Metamorphic / Polymorphic Malware-Inspired

def polymorphic_mutate(seed_payload: str, patch_description: str) -> list[str]:
