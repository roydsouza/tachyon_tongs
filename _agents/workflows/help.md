# 🛠️ Tachyon Tongs: Command Help

The following slash commands are available to facilitate your interaction with the security substrate:

- `/catalog`: Retrieves the **Exploitation Catalog**, showing all discovered CVEs and adversarial tactics.
- `/sentinel`: Manually triggers a **Sentinel** sweep to search for new internet-born threats.
- `/report`: Generates a comprehensive **Action Report** on substrate mutations, active tasks, and recent runs.
- `/acdc-loop`: Triggers the **Agent Centric Development Cycle** for the current task (Guide -> Generate -> Verify -> Solve).
- `/help`: Displays this command manifest.

*To use a command, simply type it in the chat.*
